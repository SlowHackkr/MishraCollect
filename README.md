# MishraCollect 🛠️

A personal bash tool for collecting user input and saving it in structured JSON format.

## 🔧 Features
- Validates email and age input
- Stores entries in clean JSON
- CLI-friendly with terminal banner
## Make it executable:
```bash
chmod +x run.sh
chmod +x collect.sh
```
## 🚀 Run It

```bash
./run.sh
